This portfolio serves as my digital identity, providing an overview of my academic background, technical skills, and career aspirations. It is designed with a focus on simplicity, responsiveness, and user experience, making it easy for recruiters and visitors to explore my profile.

- Modern and clean user interface
- Interactive Home section
- About Me section
- Education timeline
- Technical Skills section
- Soft Skills section

- Smooth scrolling navigation
- Well-structured and maintainable code
